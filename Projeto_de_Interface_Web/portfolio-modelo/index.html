A Atividade


A atividade proposta tem por objetivo apresentar a evolução do layout de uma capa utilizando os diversos conceitos ligados ao Design, como por exemplo: cores, fontes, formas, alinhamento, repetição, entre outros. Os alunos tiveram a oportunidade de criar um trabalho incremental à medida que os conceitos eram apresentados, até chegar à versão final.
